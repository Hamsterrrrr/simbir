# Как запустить проект

## Локально

### Установка библиотек
pip install -r requirements.txt

### Запуск Selenium Grid
java -jar selenium-server-4.28.1.jar standalone

### Запуск тестов
pytest

### Генерация отчета
allure generate

### Открытие отчета
allure serve allure-results

## Через Docker

### Запуск Docker контейнеров
sudo docker-compose up --build

### Открытие отчета
http://localhost:5050
